# My React Trips Dashboard  

**Description**  
A React + Tailwind + React‑Router based dashboard to manage trips and queries.  

## Features  
- Navigation using React Router with nested routes   and Naviagate
- Dashboard layout with sidebar and dynamic content  
- Add New Query page  
- Pages: New Query, In Progress, Converted, On Trip, Past Trips, Cancelled, Dropped, All



## Tech Stack  
React, React Router, Tailwind CSS,


## Backend / Data storage

This project uses Supabase as backend:

- On the “New Query” page — user data is saved to Supabase.  
- The app fetches and displays saved data (e.g. in “In Progress”, “Converted”, etc.) from Supabase database.
